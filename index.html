Mis dos paginas favoritas para resolver dudas de programación

  • La primera pagina que pondria aqui es Stackoverflow, ahi puedes hacer preguntas y otros usuarios pueden responderte y intentar resolver tus dudas. Es una comunidad de programadores. Hay en Ingles y Español.

  • Esta es una de las mejores paginas para preguntar/resolver dudas de programación, la pagina web se llama W3Schools. En esa pagina hay cursos de Html, Css, (Javascript), Python, etc. Tambien hay una parte de "Hazlo tu mismo" para probar los codigos lo que hace que sea muy facil de aprender y entender el concepto. Esta pagina solo tiene una version en Ingles.

    Una manera facil y divertida para aprender a programar

    Si quieres aprender a programar de una manera facil y divertida aqui te traigo la pagina perfecta para ti. Code Combat Es un juego que consiste en recoger gemas mientras guias a tu personaje apartir de codigos, con Code Combat puedes aprender: Python, Javascript, CoffeeScript y C++ (Si quieres jugar con C++, necesitas tener la version Premium), y a base de escribir codigos vas ganando gemas y niveles, y cada vez los niveles son más dificiles. Puedes jugar con tu clase si eres profesor o jugar con amigos.
